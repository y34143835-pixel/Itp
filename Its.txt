

Please follow this order:

1. File: settings.gradle.kts (In the main root folder)

This is the most important file for fixing the "Project not linked" error. Open
it, delete everything, and paste this:

pluginManagement {
    repositories {
        google()
        mavenCentral()
        gradlePluginPortal()
    }
}
dependencyResolutionManagement {
    repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
    repositories {
        google()
        mavenCentral()
        maven { url = uri("https://jitpack.io") } // Needed for some VCam libraries
    }
}

rootProject.name = "TitanInfinityPro"
include(":app")

2. File: build.gradle.kts (In the main root folder)

This is the one you showed in your screenshot. It should only look like this:

// Top-level build file where you can add configuration options common to all sub-projects/modules.
plugins {
    id("com.android.application") version "8.2.2" apply false
    id("org.jetbrains.kotlin.android") version "1.9.24" apply false
    id("kotlin-kapt") version "1.9.24" apply false
}
